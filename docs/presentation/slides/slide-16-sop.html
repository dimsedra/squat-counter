Langkah 7 — Standarisasi Operasional (SOP)

Pencegahan Masalah Berulang (SOP Penggunaan)

Berdasarkan seluruh temuan eksperimen, dirumuskan 4 aturan standar operasional untuk mencegah eror terulang:

1. Posisi Kamera Ideal
Wajib menempatkan kamera pada sudut Samping (90°) atau Serong (45°) jarak 1.8–2.0 m. Hindari posisi depan.
2. Interactive Setup Check
Sistem wajib memeriksa visibilitas landmark sendi (≥ 0.5) dan framing sebelum sesi dimulai.
3. Pakaian Olahraga Pas
Pengguna dianjurkan menggunakan pakaian pas (fitted) untuk menghindari oklusi kain pada sendi lutut.
4. Penggunaan Koordinat 2D
Gunakan 2D world coordinates murni dan abaikan kanal kedalaman z monokular yang tidak stabil.
