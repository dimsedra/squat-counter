Fleksibilitas Eksekusi: Dual Mode Deployment

Arsitektur 5-Layer ini mendukung dua mode penggunaan fleksibel tanpa perubahan pada modul kalkulasi utama:

Live Web Client-Server Mode
Deteksi pose di browser pengguna (MediaPipe JS), lalu mengirim koordinat terkompresi (~1–2 KB/frame) via WebSocket ke server untuk dihitung secara real-time.
Headless Batch Processor Mode
Pengolahan paralel multi-worker headless script untuk merekapitulasi seluruh folder video eksperimen secara otomatis ke 1 CSV summary.
